What is this?

As a student of Hack Reactor, I was always impressed when I looked around and saw the sea of colorful editors around me. This site is an attempt to visualize some of these stats and gain some insight on what goes down at 944 Market.

How?

The GitHub API allows you to pull 'statistics' from their profile

Why?

For the lulz mostly. These statistics should definitely not be taken as a serious attempt to show off or otherwise prove any sort of badassness. Comparing lines of code is like comparing something else equally as useless. We're just having some fun here
